# 02. 기능 요구사항

## MVP 기능

### 데이터 적재

- CSV 파일 업로드
- 파일 메타데이터 저장
- 원본 행 JSON 저장
- 컬럼 파싱
- 건물/층/구역/장치 자동 생성
- 측정값 저장
- 1시간 단위 집계 생성

### 모니터링

- 전체 에너지 사용량 조회
- 층별 사용량 조회
- 구역별 사용량 조회
- AC/조명/플러그 부하 비중 조회
- 온도/습도/조도 평균 조회
- 피크 사용 시간 조회

### 예측

- 층별 다음 1시간 전력 사용량 예측
- 실제값과 예측값 비교
- MAE, RMSE 계산
- 예측 결과 저장

### 이상탐지

- 야간 과소비 탐지
- 예측 오차 이상 탐지
- 센서 고정값 탐지
- 센서 급변 탐지
- 이상 이벤트 저장

### 대시보드

- Overview
- Floor View
- Zone View
- Prediction View
- Anomaly View

## MVP 제외 기능

- 사용자 인증/권한
- 실시간 스트리밍
- 3D 모델링
- 외부 알림 연동
- 복잡한 딥러닝 모델
- 다중 데이터셋 통합
- MongoDB
