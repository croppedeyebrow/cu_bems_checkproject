# 00. 프로젝트 개요

## 프로젝트명

CU-BEMS 기반 스마트빌딩 에너지 예측·이상탐지 디지털트윈 대시보드

## 한 줄 정의

CU-BEMS 스마트빌딩 데이터를 활용해 층·구역·설비 부하·실내환경 센서 데이터를 통합 관리하고, 에너지 사용량 예측과 이상탐지를 제공하는 운영형 대시보드 서비스.

## 프로젝트 목적

- 건물, 층, 구역별 에너지 사용량 모니터링
- AC, 조명, 플러그 부하별 전력 사용량 분석
- 온도, 습도, 조도와 전력 사용량의 관계 분석
- 다음 1시간 또는 다음 일자의 에너지 사용량 예측
- 야간 과소비, 예측 오차 이상, 센서 이상 탐지
- 운영자가 확인할 수 있는 React 대시보드 구현

## 프로젝트 성격

이 프로젝트는 단순 EDA가 아니라 다음을 포함하는 서비스형 프로젝트이다.

```text
데이터 적재
→ DB 설계
→ 집계 처리
→ 예측 모델
→ 이상탐지
→ API
→ 대시보드
```

## 난이도

중하

분석만 하면 하, 운영형 서비스로 구현하면 중하로 본다.

## 핵심 어필 포인트

| 역량 | 표현 방식 |
|---|---|
| 데이터 엔지니어링 | CSV 적재, Raw/Core/Aggregation 계층 구성 |
| DB 설계 | 건물-층-구역-장치-측정값 구조 |
| 백엔드 | FastAPI 기반 API |
| ML | 에너지 예측, 이상탐지 |
| 프론트엔드 | React 운영 대시보드 |
| 도메인 이해 | BEMS, 스마트빌딩, 에너지 운영 |
| 디지털트윈 감각 | 실제 운영 데이터를 가상 상태판에 반영 |
